## 🎯 このドキュメントの目的

このドキュメントでは、TechGear Storeの各ページのレイアウトを設計します。

**テキストベースのワイヤーフレームで、各ページの構成を明確にします。**

---

## 📱 レスポンシブ対応の基本方針

### デバイス別レイアウト

```
【モバイル（< 640px）】
- 1カラムレイアウト
- ハンバーガーメニュー
- 縦スクロール重視

【タブレット（640px - 1024px）】
- 2カラムレイアウト（商品グリッド）
- 一部メニュー表示

【デスクトップ（> 1024px）】
- 3-4カラムレイアウト
- フルメニュー表示
- サイドバー活用
```
