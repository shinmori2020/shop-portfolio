## 🎯 このドキュメントの目的

**特定商取引法（特商法）**に基づく表記を定義します。

### 特定商取引法とは

消費者を守るための法律で、以下を義務付けています：
- 事業者の明示
- 商品・サービスの明確な説明
- 返品・キャンセルポリシーの明示
- 消費者の権利保護

### 対象

- 通信販売（インターネット販売）
- 訪問販売
- 電話勧誘販売
- 連鎖販売取引
- 特定継続的役務提供
- 業務提供誘引販売取引

当ストアは「通信販売」に該当します。

## 9. 返品・交換について

### 返品・交換が可能な場合

#### お客様都合による返品

```
【条件】
・商品到着後7日以内
・未開封・未使用
・商品に傷や汚れがないこと

【返送料】
お客様負担

【手順】
1. お問い合わせフォームまたはメールでご連絡
2. 返品受付番号を発行
3. 商品を指定の住所に返送
4. 商品到着確認後、7〜14営業日以内に返金
```

#### 返品不可の商品

以下の商品は返品をお受けできません：
- 開封済み・使用済みの商品
- お客様の責任で商品が破損・汚損した場合
- 商品到着後8日以上経過した商品
- セール品、アウトレット品など、返品不可と明記された商品
- 食品、化粧品など衛生上の理由により返品不可の商品

#### 不良品・誤配送の場合

```
【条件】
・商品到着後7日以内にご連絡

【対応】
・無償で交換または返金
・返送料は当社負担

【手順】
1. 商品の状態が確認できる写真を撮影
2. お問い合わせフォームまたはメールでご連絡
3. 不良箇所の詳細をお知らせください
4. 代替品の発送、または返金処理
```

### 返金方法

| お支払い方法 | 返金方法 |
|------------|---------|
| クレジットカード | カード会社経由で返金（1〜2ヶ月後） |
| コンビニ決済 | 銀行振込で返金 |
| 銀行振込 | 銀行振込で返金 |

※銀行振込での返金時、振込手数料は当社負担

## 12. 不良品・破損品について

### 不良品の定義

以下の場合を不良品とします：
- 商品に明らかな傷や汚れがある
- 正常に動作しない
- 部品が不足している
- 商品説明と大きく異なる

### 対応方法

```
【交換】
同一商品の在庫がある場合、無償で交換

【返金】
同一商品の在庫がない場合、全額返金

【修理】
メーカー保証がある場合、メーカーでの修理対応
```

### ご連絡期限

```
商品到着後7日以内

※期限を過ぎた場合、対応できない場合があります。
```

## 13. 商品の欠品・在庫切れについて

### 在庫管理

当ストアでは、複数の販売チャネルで在庫を共有しているため、ご注文のタイミングにより在庫切れとなる場合があります。

### 欠品時の対応

```
1. メールでご連絡
2. 以下のいずれかをお選びいただけます：
   
   A) 次回入荷まで待つ（入荷予定日をお知らせ）
   B) 注文をキャンセル（全額返金）
   C) 類似商品への変更（差額が生じる場合は調整）
```

### 一部商品が欠品の場合

```
【分割配送】
在庫のある商品を先に発送し、欠品商品は入荷次第発送

【一括配送】
すべての商品が揃ってから発送

※ご希望の配送方法をお選びいただけます。
```
