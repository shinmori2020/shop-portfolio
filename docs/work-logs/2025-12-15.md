### 4. ProductList フィルターに固定カテゴリリストを設定

**対応箇所:**
- `src/pages/ProductList/ProductList.tsx`

**実施内容:**
- カテゴリ一覧を固定リストとして定義
- 商品データから取得したカテゴリとマージ
- 商品がないカテゴリでもフィルターに表示される仕様に変更

**実装コード:**
```typescript
const categories = useMemo(() => {
  const fixedCategories = [
    'ワイヤレスイヤホン',
    'スマートウォッチ',
    'Bluetoothスピーカー',
    'モバイルバッテリー',
    'ケーブル・アクセサリー',
    'ヘッドホン',
    'キーボード',
    'マウス',
    'Webカメラ',
    'タブレット',
    'ノートパソコン'
  ];
  const productCategories = Array.from(new Set(products.map(p => p.category)));
  const allCategories = Array.from(new Set([...fixedCategories, ...productCategories]));
  return ['all', ...allCategories];
}, [products]);
```
